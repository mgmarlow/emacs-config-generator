Emacs Quick Start

An opinionated configuration generator to help kickstart your future hacking.

Why?

There's been some recent excitement around Emacs 29 and all of its fancy new features (LSP, tree-sitter, etc.), but learning Emacs can be a long journey. It doesn't help that the Emacs defaults are likely very different from your experience with other text editors. This tool seeks to remedy that new user experience with an opinionated configuration based on your preferences.

The generated file is also heavily annotated with comments to help explain some basic Emacs Lisp esoterica and guide your exploration. This is by no means an Emacs Lisp tutorial; the best tutorial is the one that ships within Emacs itself. That said, rocking your own configuration is a great way to get started.
