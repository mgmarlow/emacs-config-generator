Emacs configuration builder

An opinionated configuration to help kick-start your hacking.

Why?

Emacs is great and more people should try it. Jumping into Doom seems like a smart way to cut past the learning curve, but I think it's better to start your own configuration and learn Emacs Lisp without introducing additional layers of complexity.

This form helps kickstart your hacking with a modern-feeling configuration while only relying on a single file of Emacs Lisp. It assumes Emacs 29 (the soon-to-be latest version) and takes advantage of many of its new features. The generated file is also heavily annotated with comments to help explain some basic Emacs Lisp esoterica to guide your exploration.

What's in the config?

Along with your selected preferences, you get:

  • Opinionated defaults that incorporate lots of goodies from Phil Hagelberg's better defaults, among other hand-picked settings.
  • Feature packages that augment the Emacs baseline with extra functionality, including Vertico, Corfu, and Marginalia.
  • Language support for many programming languages not supported OOTB.
  • Code comments that annotate all the important stuff to help you hack the config to your taste.

Get started

First, download Emacs 29 (the version matters). Then fill out the form below and generate your config file. Take that file and drop it into your Emacs home directory. If you don't know where that is, use ~/.config/emacs/init.el.
