Emacs configuration builder

An opinionated configuration to help kickstart your hacking.

Why?

Emacs is great and more people should try it. Jumping into Doom seems like a smart way to cut past the learning curve, but I think it's better to start your own configuration and learn Emacs Lisp without introducing additional layers of complexity.

This form helps kickstart your hacking with a modern-feeling configuration while only relying on a single file of Emacs Lisp. It assumes Emacs 29 (the soon-to-be latest version) and takes advantage of many of its new features. The generated file is also heavily annotated with comments to help explain some basic Emacs Lisp esoterica to guide your exploration.

What do I do after I download the file?

Drop it into your Emacs home directory as init.el, e.g. ~/.emacs.d/init.el. More information is available in the manual.
