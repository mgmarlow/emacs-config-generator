Emacs 29 Config Generator

Generate a starter configuration for Emacs with a bunch of the new fancy features in Emacs 29.

Why?

There's been some recent excitement around Emacs 29 and all of its fancy new features (LSP, tree-sitter), but learning Emacs is a long journey. This tool helps get you started without sacrificing the modern defaults you've come to expect from something like VSCode.

The generated configuration is also heavily annotated to help kickstart your hacking. A simple start to your own custom configuration in one Emacs Lisp file.
